# **App Name**: CeloRise

## Core Features:

- Impact Registry: Allow projects focused on finance, climate, education, or health to register, providing details and impact metrics.
- User Wallet Integration: Integrate Valora, MetaMask, or CeloExtensionWallet for seamless Celo blockchain interaction.
- CELO Staking for Impact: Allow users to stake CELO to support registered impact projects. Distribute rewards based on project performance. LLM will act as a tool for distributing awards among impactful projects.
- Low-Fee Transactions: Optimize gas usage in smart contracts and backend to ensure transactions are low-fee.
- Mobile-First Design: Design a responsive UI optimized for basic smartphones.
- Smart Contract Interaction: Enable users to interact with deployed smart contracts for staking, rewards, and project registration.
- Transaction Relayer: Backend route for relaying transactions to minimize user gas fees.

## Style Guidelines:

- Primary color: Soft, earthy green (#8FBC8F) to represent growth, nature, and sustainability, aligning with Celo's environmental goals.
- Background color: Very light desaturated green (#F5F5F5) for a clean and accessible feel.
- Accent color: Warm terracotta (#E07A5F) to highlight key actions and information, adding an element of trustworthiness and human connection.
- Body and headline font: 'PT Sans', a humanist sans-serif combining modern and warm elements, ensuring readability on basic smartphones.
- Use minimalist icons to represent projects, staking, and rewards, focusing on clarity and ease of understanding.
- Prioritize a clean, intuitive layout with clear information hierarchy to make the dApp accessible to users with limited digital literacy.
- Use subtle animations and transitions to enhance user experience, providing feedback without overwhelming basic smartphones.
